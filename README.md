# Conta_Banco

Um projeto simples em Java que simula o funcionamento de uma **conta bancária** com operações básicas de CRUD(CREATE, READ, UPDATE, DELETE).
Inclui o framework Spring Boot para a utilização dos endpoints(GET, POST, PUT, DELETE).

## 📌 Sobre

Este repositório contém um programa em Java que demonstra conceitos básicos de Programação Orientada a Objetos (POO), como:

- Classes e atributos
- Métodos para manipular dados
- Interação simples com o usuário

## 🚀 Como usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/enthonefs/Conta_Banco.git
   
2. Abra o projeto em um editor compatível com Java (por exemplo, Intellij ou VS Code).

3. Compile e execute o programa.
